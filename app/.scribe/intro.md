# Introduction

Lab ERP System API - Complete laboratory management system with patient management, test tracking, billing, and reporting capabilities.

<aside>
    <strong>Base URL</strong>: <code>http://localhost:8000</code>
</aside>

    This documentation aims to provide all the information you need to work with our Lab ERP System API.

    ## Features
    - **Patient Management**: Register and manage patient information
    - **Test Management**: Create and track laboratory tests
    - **Billing & Invoices**: Handle payments and generate invoices
    - **Sample Tracking**: Monitor sample collection and processing
    - **Reports**: Generate test results and financial reports
    - **User Management**: Role-based access control

    ## Authentication
    This API uses session-based authentication. You need to be logged in to access most endpoints.

    <aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).
    You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).</aside>
